üë§ User Commands

Commands available to all users.

/start User

Start the bot and get a welcome message with basic info.

/verify User

Begin biometric verification. The bot sends a QR and deep link to complete in Mercle.

Flow:
  • Bot sends QR + button
  • User scans or taps to open Mercle
  • Complete face scan; bot unmutes/allows chat
/status User

Check your current verification status and ID.

/help User

Show the interactive help message with shortcuts.

üëÆ Admin Commands

Commands for group admins (and trusted roles).

Moderation

/kick @user [reason] Admin

Kick a user from the group.

Usage:
  • Reply with /kick
  • /kick @username Spamming
/ban @user [reason] Admin

Ban a user (until manually unbanned).

/unban @user Admin

Remove an existing ban.

/mute @user [reason] Admin

Mute a user (no messages). Use /unmute to restore.

/unmute @user Admin

Unmute a user and restore chat permissions.

Warnings

/warn @user [reason] Admin

Issue a warning. The bot auto-kicks at the configured limit (default 3).

/warns @user Admin

Show recent warnings for a user.

/resetwarns @user Admin

Clear all warnings for a user.

Whitelist

/whitelist Admin

Show the current whitelist for the group.

/whitelist add @user [reason] Admin

Add a user so they bypass verification.

/whitelist remove @user Admin

Remove a user from the whitelist.

Notes & Filters

/save name content Admin

Save or update a note. Users can call it with #name or /get name.

/get name Admin

Retrieve a saved note.

/notes Admin

List all saved notes.

/clear name Admin

Delete a note.

/filter keyword response Admin

Create an auto-response filter for a keyword.

/filters Admin

List active filters.

/stop keyword Admin

Remove a filter by keyword.

Greetings & Rules

/setwelcome message Admin

Set the welcome message. Supports placeholders {name}, {mention}, {group}, {count}.

/welcome Admin

Show the current welcome message and toggle on/off.

/setgoodbye message Admin

Set the goodbye message for members leaving.

/rules User

Show the group rules if set.

/setrules text Admin

Set or update the group rules.

Settings

/settings Admin

View current group settings.

Examples:
  • /settings timeout 240 (seconds)
  • /settings action kick
  • /settings antiflood 15
  • /settings welcome off
  • /settings verify off
